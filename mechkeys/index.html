🔒 Privacy first, always

MechKeys detects when keys are pressed, never which keys. It's fully offline — zero data collection, zero analytics, zero network access. The Accessibility permission it asks for is only used to know that a key went down, so it can play a sound.

Install & requirements

📦 Installation

  1. Download and unzip MechKeys.zip
  2. Drag MechKeys.app to Applications
  3. Launch and grant Accessibility permission
  4. Click the keyboard icon in your menu bar — done!

💻 Requirements

  • macOS 14.0 (Sonoma) or later
  • Apple Silicon or Intel Mac — Universal Binary
  • Accessibility permission (to detect keystrokes)
