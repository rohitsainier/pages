What's included & what you'll need

📦 What's included

  • InstaGrab.app — Apple Silicon native (M1 / M2 / M3 / M4)
  • Signed with Developer ID, notarized by Apple
  • Material Design 3 dark UI
  • Configurable save location
  • Lifetime updates for v1.x
  • 30-second setup guide built into the app

💻 Requirements

  • macOS 11 Big Sur or later
  • Apple Silicon Mac (M1 / M2 / M3 / M4)
  • Your own Instagram account — read-only, for authentication
  • Works on public profiles and private profiles you follow
  • Posts only — Stories and Reels not supported
  • Intel Mac build available on request

🔒 Your password is never stored

Instagram requires login to access profile data. InstaGrab uses a session cookie from your Chrome browser — your password is never typed or stored anywhere. Apple-signed, notarized, and launches instantly with no “app from an unidentified developer” warnings. After purchase you'll get the signed DMG installer plus the setup guide.
