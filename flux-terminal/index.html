Features

A terminal that thinks, shares, and protects.

Flux Terminal combines the power of a native terminal with AI intelligence, peer-to-peer networking, and professional security tools.

AI

AI Command Translation

Describe what you want in plain English. Flux translates it to shell commands with danger detection and one-click execution. Supports Ollama, OpenAI, and Claude.

P2P

BharatLink File Sharing

Sovereign P2P file and text sharing built in. No servers, no accounts. Chat-style UI with drag & drop, image previews, read receipts, and encrypted transfers.

NET

28 NETOPS Tools

Full network operations dashboard. Port scan, DNS lookup, WiFi analysis, SSL inspection, threat intelligence, vulnerability scanning, and more.

GPU

GPU Accelerated

Hardware-accelerated rendering with CRT scanlines, matrix rain, keystroke particles, and holographic effects. Buttery smooth at every frame.

MCP

MCP Server Support

Model Context Protocol integration lets AI access your tools, files, and APIs through a standardized interface. Extend AI capabilities infinitely.

SSH

SSH + Multi-Tab

Manage multiple terminal sessions with tabs. SSH into remote servers. Command palette for instant access to everything. All keyboard-driven.

SNIP

Snippet Library

Save, organize, and instantly recall command snippets. Categories, search, import/export. Never forget a useful command again.

CFG

Hot-Reload Config

Configuration stored in your OS config directory. Changes apply instantly without restart. Customize everything from themes to AI providers.

OSS

Open Source

100% open source, built with Rust and Tauri 2.0. Fast, secure, and cross-platform. Contribute or build your own extensions.
